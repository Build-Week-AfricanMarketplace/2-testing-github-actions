# Date picker

Date and time pickers allow users to select a single or a range of dates and times.

## Installation

There are two ways to install and use this component, this can be done with NPM:

```sh
npm install -S @aileron/date-picker
```

or via cdn by:

```html
<script defer src="https://cdn.aa.com/aileron/date-picker@latest/dist/date-picker.js"></script>
```
