# Accordion

## Installation

There are two ways to install and use this component, this can be done with NPM:

```sh
npm install -S @aileron/accordion
```

or via cdn by:

```html
<script defer src="https://cdn.aa.com/aileron/accordion@latest/accordion.js"></script>
```
