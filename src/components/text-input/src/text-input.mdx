# Text Input

## Installation

There are two ways to install and use this component, this can be done with NPM:

```sh
npm install -S @aileron/text-input
```

or via cdn by:

```html
<script defer src="https://cdn.aa.com/aileron/text-input@latest/dist/text-input.js"></script>
```
