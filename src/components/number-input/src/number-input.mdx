# NumberInput

## Installation

There are two ways to install and use this component, this can be done with NPM:

```sh
npm install -S @aileron/number-input
```

or via cdn by:

```html
<script defer src="https://cdn.aa.com/aileron/number-input@latest/dist/number-input.js"></script>
```
