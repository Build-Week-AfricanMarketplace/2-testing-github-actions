# Checkbox

## Installation

There are two ways to install and use this component, this can be done with NPM:

```sh
npm install -S @aileron/checkbox
```

or via cdn by:

```html
<script defer src="https://cdn.aa.com/aileron/checkbox@latest/checkbox.js"></script>
```
